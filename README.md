# Dracula - Markdown Editor

## Preparation
You need at least Node 12 (we recommend Node 16) and [yarn](https://yarnpkg.com/).

## Development mode

1. Clone this repo
2. Go inside the repo
3. Run `yarn`
4.Run
    - `yarn start`
